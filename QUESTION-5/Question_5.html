5. WHAT ARE HTML ENTITIES? EXPLAIN EXAMPLES OF TWO ENTITIES.


  DEFINITION:

  Some words are reserved in HTML.
  If you want to display reserved character, you have to use HTML Entities.
  In short, HTML reserved characters are replaced with HTML Entities.
  There are Entity names and Entity number to display reserved character.
  Entity Number is easy to remember than Entity Name.

  EXAMPLE:

Description Entity Name Entity Number Result
non-breaking space &nbsp; &#160;
less than &lt; &#60; <
greater than &gt; &#62; >
