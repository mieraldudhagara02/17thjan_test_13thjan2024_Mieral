6. WHAT DO YOU UNDERSTAND BY UNIVERSAL SELECTOR? EXPLAIN WITH EXAMPLE.


  > Universal Selector selects all the elements of the HTML page.
  > It is written as an asterisk "*".
  > Universal Selector is used for selecting all elements.
  > When you want to put a specific style to all the HTML Elements, Universal Selector is becomes helpful.

  SYNTAX:

    * {
        property : "value"
      }

  EXAMPLE:

    * {
        color : #264348
      }

  Here,
    I'm using Universal Selector for all the HTML Elements to display same color of fonts.
