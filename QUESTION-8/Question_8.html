8. WHAT IS MEANT BY RGB STREAM? SET A BACKGROUND COLOR OF WEBPAGE USING RGB STREAM.


  > RGB represents mainly three color.
    1. R : Red
    2. G : Green
    3. B : Blue
  > Each component can have a value between 0 and 255. Thus, there is total 256 colors.

  > If you want display the black color, set all parameters to 0,
    Like, rgb(0, 0, 0);

  > If you want display the white color, set all parameters to 255,
    Like, rgb(255, 255, 255);

  > Here, background color of webpage is set by using rgb stream.
    body {
            background-color : rgb(207, 248, 255);
          }

  EXAMPLE:

rgb(255, 0, 0)
rgb(0, 255, 0)
rgb(0, 0, 255)
