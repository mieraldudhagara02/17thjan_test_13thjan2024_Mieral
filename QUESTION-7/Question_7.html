7. WHAT ARE THE ELEMENTS OF THE CSS BOX MODEL? EXPLAIN WITH EXAMPLE.


  The CSS Box Model is a Container.
  It contains multiple elements like 'margin, padding, border and content'.

  1. Content
    Contains the data in form of text, images.
    We can be sized of the content by using Height/Width.

  2. Padding
    Padding is used to create space around the element, inside any defined border.

  3. Margin
    Padding is used to create space around the element, outside any defined border.

  4. Border
    Border is used to cover the content & padding.

  EXAMPLE:

M I R A L
  Here,
    > Text 'M I R A L' is the content.
    > Space around the text 'M I R A L' and inside the border is called Padding.
    > I have added 50 px light sea green color border.
    > Space around the border and inside the main content is called Margin.
