4. WHAT ARE VOID ELEMENTS IN HTML? EXPLAIN WITH EXAMPLES OF 5 VOID ELEMENTS.


  DEFINITION:

  Element : Opening Tag + Content + Closing Tag
  Void Elements only have Starting Tag.
  Void Elements do not contain any content within them.
  VOID ELEMENTS
    > do not have closing tag
    > do not contain any content within them
    > can not be nested

  EXAMPLE:

  1. <br>
  2. <img>
  3. <hr>
  4. <link>
  5. <meta>
