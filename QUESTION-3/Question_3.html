3. DEFINE THE LIST TYPES IN HTML. EXPLAIN WITH EXAMPLE OF NAVBAR AND APPLY A DROPDOWN IN NAVBAR ALSO.


  DEFINITION:

  In HTML, Lists are used to group related items together.

  There are three types of Lists in HTML:
    1. Unordered List
    2. Ordered List
    3. Description List

  1. Unordered List : Unordered List is used to group of related items not in particular order.
  2. Ordered List : Ordered List is used to group of related items in particular order.
  3. Description List : Description List is used to group of related items with description of each items.

  EXAMPLE:

  1. Unordered List   2. Ordered List   3. Description List

  • Ball
